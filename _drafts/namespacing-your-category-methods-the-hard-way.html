The Problem

We have been working with a third party library that provides some security infrastructure for mobile devices.  The vendor provides a static library framework that gets compiled into your app.  Among other functions a developer is able to open other secure applications on the device.  We'd had everything working flawlessly during development.  We received notification that towards the end of development with this product that users were no longer able to open other secure applications and send a file along with it.
